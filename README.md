# Open Claude VS Code Extension

AI coding assistant powered by Open Claude — chat, stream, and inject code with 23+ frontier models.

## Features
- **23 Models**: Claude Opus/Sonnet/Haiku, GPT-5, Gemini, Grok
- **Chat & Inject**: Select code in editor and inject directly to chat
- **Code Blocks**: Copy and insert buttons for all code responses
- **Glassmorphism UI**: Beautiful dark theme chat sidebar

## Usage
1. Open settings (`Ctrl+,`) and search for "Open Claude"
2. Enter your API Key from [https://open-claude.com/keys](https://open-claude.com/keys)
3. Select your preferred model
4. Click the Open Claude icon in the Activity Bar to start chatting.
